Every EcoItems command and the permission node it needs. Use these to give items to players, reload after config changes, and move items to and from the community library.

| Command | Description | Permission |
| --- | --- | --- |
| `/ecoitems reload` | Reload the plugin configs | `ecoitems.command.reload` |
| `/ecoitems give <player> <item> [amount]` | Give an EcoItem to a player | `ecoitems.command.give` |
| `/ecoitems import <id>` | Import an item from [lrcdb](https://lrcdb.auxilor.io/) | `ecoitems.command.import` |
| `/ecoitems export <id>` | Export an item to [lrcdb](https://lrcdb.auxilor.io/) | `ecoitems.command.export` |
